# 📖 Holy Intensive

Aplicativo mobile inspirado no estilo do Duolingo, com o objetivo de guiar o usuário na leitura completa da Bíblia de forma progressiva, simples e motivadora 📱✨

---

## 🚀 Status do Projeto
🚧 Em desenvolvimento

---

## 🎯 Objetivo

Ajudar pessoas a criarem o hábito diário de leitura da Bíblia através de:
- Progresso contínuo
- Interface simples
- Experiência gamificada

---

## 📱 Funcionalidades (atuais)

- 📖 Leitura de conteúdos bíblicos
- 🧭 Navegação entre telas
- 🎨 Interface inicial do app

---

## 🔮 Próximas funcionalidades

- ✅ Sistema de progresso diário
- 🔥 Streak (dias seguidos)
- 🧠 Quiz bíblico
- 👤 Sistema de login
- ☁️ Backup em nuvem

---

## 🛠️ Tecnologias utilizadas

- Flutter
- Dart
- Android SDK
